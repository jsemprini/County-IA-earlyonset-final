How to read the map

For percentage change, blue represents lower values and red represents higher values, centered at zero. For the two fixed periods, darker red indicates larger values.

How to use the table

The county table ranks counties from highest to lowest for the currently displayed measure. Hover over the map to see county-level values.

Data note

Counties without usable numeric values are omitted from the ranked table and may not appear with a filled color on the map. Cancer rates are reported as cancer cases per 100,000 population. Percentage Change (PC) reports the change from 2000-2011 to 2012-2022, as a proportion on a binary scale. Multiply the PC value by 100 to interpret the value as a % on a 0-100 scale.
